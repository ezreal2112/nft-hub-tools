# Contributing

Thanks for helping NFT HUB! To propose a change:
1. Open an Issue describing the change and why.
2. Fork the repo & create a feature branch: feat/<short-name>.
3. Write clear commits.
4. Add/adjust docs and examples.
5. Open a Pull Request and link the Issue.

## Code style
- Keep scripts minimal and well-commented.
- Use .env for secrets; never commit keys.

## Commit message format (recommended)
- feat: new tool or capability
- fix: bug fix
- docs: documentation only
- chore: tooling/infra
